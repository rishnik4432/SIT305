# Personal Event Planner App
A simple Android application to manage personal events, trips, or appointments.
Built with Java, Room Database, and Navigation Component.

## Features
- Add, edit, and delete events.
- View upcoming events sorted by date.
- Choose event category: Work, Social, Travel
- Pick date and time (prevents past dates)
- Data persists locally using Room database
- Bottom navigation bar for easy switching between event list and add event screen

## How To Use
- **Add and event**: Go to the "Add Event" tab, fill in the title, category, location, and date/time, then tap "Save Event".
- **View events**: The "Events" tab shows all upcoming events sorted by date.
- **Edit an event**: Tap the "Edit" button on any event card.
- **Delete and event**: Tap the "Delete" (a confirmation toast will appear).
