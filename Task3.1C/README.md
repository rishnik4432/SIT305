# Quiz App

A simple Android app that quizzes users on their general knowledge.

## Features
- Input field for username
- Five question with real-time progress tracking bar
- Final score and session persistence

## How to Use
1. Input username
2. Select correct answer for question
3. Submit answer and move on to the next question
4. Check your final score
5. Tap 'Start a new quiz' or 'Finish'

## Visual Feedback
- Green: Your answer is correct
- Red: Your answer is wrong; Correct answer will be shown in green

## Built With
- Android Studio
- Java
