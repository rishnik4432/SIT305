# Travel Companion App
A simple Android app that converts currency, fuel efficiency and temperature using fixed 2026 rates.

# Features
- Currency: USD, AUD
- Fuel efficiency: mpg, km/L
- Temperature: Celsius, Fahrenheit

# How to use
1. Select source and destination units
2. Enter a value
3. Tap 'Convert' to see the result

# Conversion formulas
- 1 USD = 1.55 AUD
- 1 mpg = 0.425 km/L
- Celsius to Fahrenheit: (C * 1.8) + 32

# Built with
- Android Studio
- Java
